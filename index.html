About Us

Debuggers

Cricket is a great game and we all love to play, watch, read and talk about it. This projects attempts to use machine learning to judge player performances over a series of matches, and at the same time apply this on the popular application Dream11. Cricket lovers can now finally compare their analyses of a game with machine learning analyses, and at the same time earn some credit. The matches are not just restricted to T20, ODI or only male category, but to all cricket matches of any league in general. We also intend to extend this to other games in the future, so sports fans can enjoy and learn some theory at the same time.
